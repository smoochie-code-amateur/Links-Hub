#### A handy sidebar that helps you keep all your links in one place!

## 📁 Organise links by category 
This is exactly what sets Links Hub apart from simple browser bookmarks. Thanks to flexible customisation, you can create different categories that you like or simply need.

## 📄 Create different pages with links 
It’s very handy to have links not just in one place, but to organise them into different pages. For example, you can keep links for work and personal use on separate pages. 

## 🖌️ Customise the look however you like 
By default, there are 4 themes included, plus a 5th custom one. Change the colours and rounding to suit your preferences. 
*(Note: this feature will be improved and updated).*

## 🧷 Import and export link collections 
This will help you transfer your link Hub between devices and share them with others! 

## 🫂 Open source 
The source code is on GitHub; you can fork it if needed and modify it as you wish. 

## 🔐 Private and no tracking 
I don’t need your data, I don’t collect it and I don’t track you in any way. Your links and information are private. All preferences are stored locally in your browser.

## 💻 Installation

**Available on:** [Firefox](https://github.com/smoochie-code-amateur/Links-Hub/blob/main/README.md#-firefox) | [Chromium](https://github.com/smoochie-code-amateur/Links-Hub/blob/main/README.md#-chromium)

#### 🦊 Firefox 
You can install it from the [add-ons store](https://addons.mozilla.org/uk/firefox/addon/links-hub/).

#### 🔵 Chromium
I don’t want to pay Google or publish the extension there. Instead, you can very easily add it to your browser yourself. 
> It’s even more private that way ;]

1. Type `chrome://extensions/` into the adress bar
2. Toggle on `Developer mode`
3. Click `Load unpacked extension`
4. Select the folder with the extension files.
5. Pin it to the toolbar for convenience.

**Done!** Feel free to use it.
